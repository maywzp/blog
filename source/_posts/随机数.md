## 函数
```javascript
Math.random()       //产生[0,1) 区域的随机数
Math.floor(num)     //返回指定数字的整数
Math.round(num)     //返回指定数字四舍五入后的整数
Math.ceil(num)      //返回大于或等于指定数字的最小整数

```

## 数字
### 返回指定范围的随机数：
```javascript
公式： Math.random()*(m-n) + n     //m为较大的数，n为较小的数据
例：返回10~20之间的随机数：
       Math.random()*(20-10)+10
```

### 返回固定长度的随机数：
```
function randNum(len){
    var rand = "";     //要返回小数 rand = "0."
    for(var i=0;i<len;i++){
        rand += Math.floor(Math.random()*10);
     }
    return rand;
}
```

function randomWord(randomFlag, min, max){
 var str = "",

 // 随机产生

 range = Math.round(Math.random() * (max-min)) + min;
